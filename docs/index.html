Clase ProcesadorUsuarios

Clase que procesa listas de usuarios y separa admins de invitados. Permite obtener un resumen de cada rol en un solo String.

Constantes

  • ROL_ADMIN – Constante que representa el rol de administrador
  • ROL_INVITADO – Constante que representa el rol de invitado

Métodos Públicos

String procesarLista(List<String> usuarios)

Procesa una lista de usuarios en formato "nombre:rol" y devuelve un String con los admins e invitados separados.

  • Parámetros: usuarios – Lista de usuarios con formato "nombre:rol"
  • Retorno: String que contiene admins e invitados

Métodos Privados

String procesarAdmin(String nombre)

Procesa un admin y devuelve su nombre formateado.

  • Parámetros: nombre – Nombre del admin
  • Retorno: Nombre del admin con coma al final

String procesarInvitado(String nombre)

Procesa un invitado y devuelve su nombre formateado.

  • Parámetros: nombre – Nombre del invitado
  • Retorno: Nombre del invitado con coma al final
